# KINGBOT Platform

This workspace now includes a working Node.js backend for:
- user signup and login
- per-user account data storage
- protected dashboard and analytics pages
- AI support endpoint
- broker connection workflow

## Run locally

```bash
npm install
node server.js
```

Then visit:
- http://127.0.0.1:3000/
- http://127.0.0.1:3000/signup.html
- http://127.0.0.1:3000/login.html
- http://127.0.0.1:3000/dashboard.html
